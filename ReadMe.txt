Front End : Used Expo Go (React Natve)
Back End: Node.js

Demo Video: https://drive.google.com/file/d/1HQW7h7sqr6Wdwp7gqIsmePSp5igZlzMd/view?usp=drive_link

How to use code:
1. Download the Code
2. Go to track-server directory and run "npm run dev" command
3. Go to some directory(track) and run "ngrok http 3000" command to make the communication
4. Go to track directory and run "npm start" command
5. Download "expo go" app from Play store/App Store
6. Scan the QR code that generated by Terminal.
7. Then enter the email and password below to login as an Admin.
          email: "admin@gmail.com"
          password:"admin1234"

Then you can create students and teacher credentials as an ADMIN then use those crdentials to log in as STUDENT and TEACHER.
