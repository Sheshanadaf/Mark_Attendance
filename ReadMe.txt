## Overview

Track App is a mobile application built using **React Native (Expo)** and **Node.js**, designed to manage Students and Teachers under an Admin role. Admins can create accounts, and Students/Teachers can log in using those credentials. The system uses **ngrok** for API tunneling and **MongoDB** as the backend database.

---

## Architecture

### Key Components

* **Frontend:** React Native (Expo)
* **Backend:** Node.js + Express.js
* **Database:** MongoDB
* **API Tunneling:** ngrok
* **Mobile Runtime:** Expo Go App
